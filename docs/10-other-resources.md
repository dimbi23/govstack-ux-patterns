# 10. Other Resources

## GovStack AI Readiness Guide
The strategic context for this specification. Describes the shift from portal-based to agent-based government infrastructure and the imperative for deterministic, machine-readable systems.
- URL: https://specs.govstack.global/ai-readiness

## GovStack Service Patterns
The source specification for all 11 step patterns and 12 functional patterns encoded in this repository.
- URL: https://specs.govstack.global/service-patterns

## GovStack Implementation Playbook
Guidance for countries implementing GovStack-compliant digital services, including service design methodology.
- URL: https://app.gitbook.com/s/DLeKag4x9xUV26hiPlNo/

## GovStack Building Block Specifications
Specifications for the infrastructure building blocks referenced in patterns (identity, payments, consent, messaging, etc.).
- Identity BB: https://identity.govstack.global
- Payments BB: https://payments.govstack.global

## DPI Safeguard Framework — UN 2024
The United Nations framework for Digital Public Infrastructure safeguards, including the principle of effective citizen redress referenced in the DPI context of this work.
- URL: https://www.undp.org/digital/dpi-safeguards

## JSON Schema (draft 2020-12)
The specification for the schema format used in this repository.
- URL: https://json-schema.org/draft/2020-12
